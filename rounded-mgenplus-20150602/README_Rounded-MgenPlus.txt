Version 1.059.20150602


■ このフォントについて

Rounded Mgen+ (ラウンデッド ムゲンプラス) は、フリーフォントである
「M+ OUTLINE FONTS」をベースに丸ゴシック風に加工し、
さらにこれに含まれない漢字・記号のグリフを、同様に丸ゴシック風に加工した
「源ノ角ゴシック (Noto Sans CJK / Source Han Sans の日本語部分)」で
補った合成フォントです。


このバージョンでは、M+ OUTLINE FONTS 2015 年 5 月 31 日時点での CVS 版をベースとしています。
TESTFLIGHT-059 の漢字に加え、新たに制作された以下の漢字を含んでいます。

窩筵箒紆縋网羹羮耄臀舅艀艱蒟蒻薨藉虱蜃袱褥褻襾訛詈諱讒讀贖續瀆躓辜逅邂邁陋飩饂馭
髏髑躅觸鬥鯱鴦鹵麥黹齧齪齷龜龠秭𥝱禸辵沪舒俘剽吝嗇堡塹宸嶼檄歿狆

M+ 由来の漢字グリフは 4,972 字がこのフォントに収録されており、
それ以外の漢字は源ノ角ゴシックがベースとなっています。


詳細は、以下のサイトをごらんください。
http://jikasei.me/font/rounded-mgenplus/


■ 各フォントの違いについて

アーカイブには多くのフォントファイルが含まれますが、ほんのわずかな違いしかないものもあります。
すべてインストールすると 213 ファイル、1.65 GB あり、システムが重くなる可能性があります。
まずは「rounded-mgenplus-1c」から始まるフォントのみ、インストールして使い始めてみてください。
具体的な違いが分かり、使い分けが可能になった場合は、目的に合わせて適宜取捨選択してみてください。


【X, L について】

「Rounded-X Mgen+」は、文字の丸める処理を強くしたものです。
「Rounded-L Mgen+」は、文字の丸める処理を弱くしたものです。


【1c, 2p などの違いについて】

「1」と「2」では、かなの形状、漢字のサイズ (「2」は「1」の 98%) が異なります。
英数字も、一部文字が、それぞれのかなにあわせたデザインになっています。

・1 ... 直線処理と柔らかな曲線の対比が特徴的
・2 ... 従来的なゴシック体の雰囲気を残しつつ現代的な表情も併せ持つ

「p」「c」「m」「mn」では、英数字の形状が異なります。

・p ... 明るく伸びやかなデザインを心がけた
・c ... 文字組みの時のまとまりの良さを意識して再設計された
・m ... 等幅。識別性を重視しつつ素直な字形を目指した
・mn ... 等幅。コーディングのための新しいデザイン性を意識した (1 のみ)

「pp」「cp」は、「p」「c」のひらがな、全角英数字、一部記号 (約物) などの文字幅を
「MS Pゴシック」のようにプロポーショナル (文字の形に合わせて可変) にしたものです。


【文字の太さについて】

7 種類のウエイト (文字の太さ) があります。

・thin : 極細 (「m」「mn」にはありません)
・light : 細字
・regular : 標準
・medium : 中太
・bold : 太字
・heavy : 特太
・black : 極太 (「m」「mn」にはありません)


■ ライセンスと著作権について

・Rounded Mgen+ は、M+ OUTLINE FONTS にないグリフを源ノ角ゴシックで補ったフォントです。
・フォントデータに含まれる、源ノ角ゴシック由来の文字グリフの著作権は Adobe が所有しています。
・フォントデータに含まれる、M+ OUTLINE FONTS 由来の文字グリフの著作権は M+ FONTS PROJECT が所有しています。
・Megn+ のフォントファイルは、源ノ角ゴシックと同じ SIL Open Font License 1.1 のもとで使用することができます。
・ただし、使用や埋め込み・改変を行った結果、使用したのが M+ OUTLINE FONTS 由来の文字グリフのみだった場合は、
  より要件が少ない M+ FONT LICENSE を適用することができます。

SIL Open Font License 1.1 の内容は、アーカイブに同梱の SIL_Open_Font_License_1.1.txt に記載されています。
この日本語訳は、以下から参照することができます。
http://osdn.jp/projects/opensource/wiki/SIL_Open_Font_License_1.1

M+ OUTLINE FONTS のグリフは、同梱のファイル LICENSE_J に記載された自由な
M+ FONTS LICENSE に基づき使用しています。
